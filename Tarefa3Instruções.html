Bola 8 Mágica

Você aprendeu uma ferramenta poderosa em JavaScript: controle de fluxo! Na verdade, é tão poderoso que pode ser usado para prever a sorte de alguém.

Neste projeto vamos construir uma Bola Oito Mágica .

O usuário poderá inserir uma pergunta, então nosso programa produzirá um resultado de sorte aleatório.

  1. Na primeira linha do programa, defina uma variável chamada userName que seja definida como uma string vazia. Se o usuário quiser, pode inserir seu nome entre as aspas.
  2. Abaixo desta variável, crie uma expressão ternária que decida o que fazer se o usuário inserir um nome ou não. Se o usuário inserir um nome - como 'Jane'- use a interpolação de string para fazer logon Hello, Jane! no console. Caso contrário, basta exibir Hello!
  3. Crie uma variável chamada userQuestion. O valor da variável deve ser uma string que é a pergunta que o usuário deseja fazer ao Magic Eight Ball.
  4. Escreva um console.log() para o userQuestion, informando o que foi perguntado. Você pode incluir o nome do usuário na declaração console.log(), se desejar!
  5. Precisamos gerar um número aleatório entre 0 e 7.

    Crie outra variável chamada randomNumber. Atribua o seu valor com a seguinte expressão que usa dois métodos ( Math.floor() e Math.random()) da biblioteca Math.

    Math.floor(Math.random() * 8);

    Saiba mais o que essa expressão faz .
  6. Crie mais uma variável chamada eightBall, e defina-a como string vazia. Vamos salvar um valor para esta variável nas próximas etapas, dependendo do valor de randomNumber.
  7. Precisamos criar um fluxo de controle que receba o randomNumber que criamos na etapa 5 e, em seguida, atribua a eightBall uma resposta que uma Magic Eight Ball retornaria. Pense em utilizar declarações if/ else ou switch. Aqui estão 8 frases da Magic Eight Ball que gostaríamos de salvar na variável eightBall:
    • It is certain (É uma certeza)
    • It is decidedly so (É decididamente assim.)
    • Reply hazy try again (Resposta nebulosa, tente de novo)
    • Cannot predict now (Não consigo prever agora.)
    • Do not count on it (Não conte com isso.)
    • My sources say no (Minhas fontes dizem que não.)
    • Outlook not so good (Perspectiva não tão boa)
    • Signs point to yes (Os sinais apontam que sim)
    Se randomNumber for 0, atribua uma resposta para a variável eightBall; se randomNumber for 1, atribua a próxima resposta e assim por diante. Se você estiver se sentindo criativo, faça suas próprias respostas!
  8. Escreva um console.log() para exibir a resposta da Magic Eight Ball, o valor da variável eightBall.
  9. Execute seu programa algumas vezes para ver os resultados aleatórios aparecerem no console! Se você quiser prática extra:
    • Se você começou com uma switch, converta-a em instruções if/ else if / else
    • Se você começou com instruções if/ else if/ else, converta-as em uma instrução switch.
