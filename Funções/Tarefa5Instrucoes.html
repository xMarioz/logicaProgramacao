Pedra, Papel, Tesoura

Pedra, papel, tesoura é um jogo clássico para dois jogadores. Cada jogador escolhe pedra, papel ou tesoura. Os itens são comparados e o jogador que escolher o item mais poderoso vence.

Os resultados possíveis são:

  • Pedra destrói tesouras.
  • Tesoura corta papel.
  • O papel cobre a pedra.
  • Se houver empate, o jogo termina empatado.

Nosso código irá dividir o jogo em quatro partes:

  1. Obtenha a escolha do usuário.
  2. Obtenha a escolha do computador.
  3. Compare as duas opções e determine um vencedor.
  4. Inicie o programa e exiba os resultados.

Lembre-se de comentar as principais partes do código para recordar conceitos e lembrar o que cada parte faz.


  1. O usuário deve poder escolher 'pedra', 'papel' ou 'tesoura' quando o jogo começar. Usando const e a sintaxe de arrow function crie uma função chamada getUserChoice que receba um único parâmetro chamado userInput
  2. Agora que um usuário pode passar via parâmetro palavras como 'pedra' e 'Pedra', vamos utilizar a função toLowerCase() do JavaScript para fazer com que o valor em userInput fique em letra minúscula.

    Você pode usar um código assim:

    userInput = userInput.toLowerCase().
  3. Ao obter a escolha do usuário, você também deve verificar se o usuário digitou uma escolha válida: 'pedra', 'papel' ou 'tesoura'.
    Dentro de getUserChoice(), escreva uma instrução if/ else que certifique-se de que userInput seja 'pedra', 'papel' ou 'tesoura'. Se isso acontecer, então retorne(return) userInput. Caso contrário, use console.log() para imprimir uma mensagem de erro no console.
  4. Teste a função chamando-a com entradas válidas e inválidas e imprimindo os resultados no console. Você pode excluí-lo quando souber que sua função funciona.
  5. Agora precisamos que o computador faça uma escolha. Crie uma nova função nomeada getComputerChoice em parâmetros. Dentro de seu bloco, utilize Math.random() e Math.floor() para obter um número inteiro entre 0 e 2. Então, dependendo do número, retorne 'pedra', 'papel' ou 'tesoura'.
  6. Teste a função chamando-a várias vezes e imprimindo os resultados no console. Você pode excluí-lo quando souber que sua função funciona.
  7. Agora é hora de determinar um vencedor. Crie uma função chamada determineWinner que receba dois parâmetros chamados userChoice e computerChoice. Esta função irá comparar as duas escolhas jogadas e então retornar (return) se o jogador humano ganhou, perdeu ou empatou.
    Vamos lidar com a condição de empate primeiro. Dentro da função determineWinner(), escreva uma instrução if que verifique se o parâmetro userChoice é igual ao parâmetro computerChoice. Se assim for, retorne uma string que o jogo foi um empate.
  8. Se o jogo não estiver empatado, você precisará determinar um vencedor. Comece escrevendo uma instrução if que verifica se o userChoice é 'pedra'. Dentro do bloco if da declaração, escreva outra declaração if/else. O if/else mais interno deve verificar se o computerChoice é 'papel'. Em caso afirmativo, returne uma mensagem de que o computador venceu. Se não, retorne uma mensagem de que o usuário ganhou.
  9. Depois, escreva outra instrução if para verificar se userChoice é 'papel'. Dentro da declaração if, a variável computerChoice deve ser 'tesoura' ou 'pedra'. Escreva a lógica que retornará um vencedor.
  10. Em seguida, escreva outra instrução if para verificar se userChoice é 'tesoura'.
    Dentro desta instrução if, computerChoice deve ser 'pedra' ou 'papel'. Escreva a lógica que irá retornar um vencedor.
  11. Não esqueça de testar sua função!
  12. Tudo está configurado. Agora você precisa iniciar o jogo e registrar os resultados. Crie uma função chamada playGame. Dentro da função playGame(), crie uma variável chamada userChoice e atribua o seu valor igual ao resultado da chamada getUserChoice(), passando 'pedra', 'papel' ou 'tesoura' como argumento. Crie outra variável chamada computerChoice e defina-a igual ao resultado da chamada getComputerChoice(). Em ambas as variáveis, use console.log para imprimi-las no console.
  13. Finalmente, vamos determinar quem ganhou. Dentro da função playGame(), chame a função determineWinner(). Passe as variáveis userChoice ​​e computerChoice como seus parâmetros. Certifique-se de colocar esta chamada de função dentro de uma instrução console.log() para que você possa ver o resultado. Então, para iniciar o jogo, chame a função playGame() na última linha do seu programa.
  14. Melhore o seu jogo adicionando um código de trapaça secreto. Se um usuário digitar 'bomba' como sua escolha, garanta que o usuário humano seja o vencedor, não importa qual seja a escolha do computador.
